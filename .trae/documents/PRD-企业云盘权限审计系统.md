## 1. 产品概述

企业云盘共享文件夹权限审计系统，面向行政、法务和信息安全负责人，用于全面盘点企业云盘共享文件夹的权限风险，标记问题并跟踪整改进度，支持季度内控检查的报告导出。

- 核心价值：快速识别权限风险（外部访问、离职人员、编辑人数过多、长期未访问），闭环管理整改流程，一键生成审计报告
- 目标用户：行政人员、法务专员、信息安全负责人、部门负责人

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 审计人员 | 行政/法务/信息安全负责人 | 查看风险清单、发起整改、查看审计报告 |
| 文件夹负责人 | 各部门文件夹管理员 | 接收待办、执行整改、回填结果 |

### 2.2 功能模块

1. **风险清单首页**：四类风险分类展示、搜索筛选、风险排序
2. **文件夹详情页**：成员列表、角色权限、授权来源、最近访问时间
3. **整改记录**：勾选待整改成员、填写处理意见、生成待办、进度跟踪
4. **审计报告页**：按部门/项目/文件夹维度统计、本月问题数量、已处理数量、遗留项、导出功能

## 3. 核心流程

### 3.1 审计整改流程

审计人员进入首页查看风险清单 → 选择高风险文件夹进入详情 → 勾选需要收回权限的成员 → 填写处理意见 → 提交整改申请 → 系统生成待办给文件夹负责人 → 负责人接收待办 → 执行权限调整 → 回填处理结果并上传截图 → 审计人员确认完成 → 记录归档

```mermaid
flowchart LR
    A["审计人员查看风险清单"] --> B["进入文件夹详情"]
    B --> C["勾选待整改成员"]
    C --> D["填写处理意见"]
    D --> E["提交整改申请"]
    E --> F["生成待办任务"]
    F --> G["文件夹负责人接收"]
    G --> H["执行权限调整"]
    H --> I["回填结果+上传截图"]
    I --> J["审计人员确认"]
    J --> K["记录归档"]
```

### 3.2 审计报告生成流程

选择报告周期 → 选择统计维度（部门/项目/文件夹） → 系统汇总数据 → 生成可视化图表 → 预览报告 → 导出文件

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深海蓝 (#0F3460) - 传达专业、严谨、可信
- **辅助色**：警示橙 (#E94560) - 风险标记、危险操作；成功绿 (#27AE60) - 已完成状态；警告黄 (#F39C12) - 待处理状态
- **中性色**：深灰 (#1A1A2E) 中灰 (#535C68) 浅灰 (#F5F7FA) 白色 (#FFFFFF)
- **按钮风格**：圆角 6px，主按钮深蓝色填充，悬停时略深
- **字体**：中文使用 PingFang SC / Microsoft YaHei，数字和英文使用 Inter，层级清晰
- **布局风格**：左右分栏布局，左侧导航 + 右侧内容区，卡片式内容展示
- **图标**：使用 Lucide 线性图标，简洁专业

### 4.3 响应式

- 桌面端优先设计（1280px 及以上）
- 平板端：侧边栏可折叠，内容区自适应
- 移动端：底部导航，卡片堆叠布局

### 4.4 动效设计

- 页面切换：淡入淡出 200ms
- 卡片悬停：轻微上浮 + 阴影加深
- 风险标记：呼吸动画（高风险项）
- 数据加载：骨架屏 + 淡入
